<div class="pilwrap" id="deployment">
  <h2>
    <a href="#deployment" name="deployment" class="pilcrow"></a>
Deployment
  </h2>
</div>
<p>When you're ready to deploy your NestJS application to production, there are some key steps you can take to ensure it runs as efficiently as possible. Check out the <a href="https://docs.nestjs.com/deployment">deployment documentation</a> for more information.</p>
<p>If you are looking for a cloud-based platform to deploy your NestJS application, check out <a href="https://mau.nestjs.com">Mau</a>, our official platform for deploying NestJS applications on AWS. Mau makes deployment straightforward and fast, requiring just a few simple steps:</p>
<pre><code class="bash">$ npm install -g @nestjs/mau
$ mau deploy
</code></pre>
<p>With Mau, you can deploy your application in just a few clicks, allowing you to focus on building features rather than managing infrastructure.</p>
